### Problem 2. Multiplication Sign
*	Write a script that shows the sign (+, - or 0) of the product of three real numbers, without calculating it.
*	Use a sequence of `if` operators.

_Examples:_

| a    | b    | c    | result |
|------|------|------|:------:|
| 5    | 2    | 2    | +      |
| -2   | -2   | 1    | +      |
| -2   | 4    | 3    | -      |
| 0    | -2.5 | 4    | 0      |
| -1   | -0.5 | -5.1 | -      |
