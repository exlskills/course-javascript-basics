### Problem 5. Digit as word
*	Write a script that asks for a digit `(0-9)`, and depending on the input, shows the digit as a word (in English).
*	Print `“not a digit”` in case of invalid input.
*	Use a `switch` statement.

_Examples:_

| digit | result      |
|-------|-------------|
| 2     | two         |
| 1     | one         |
| 0     | zero        |
| 5     | five        |
| -0.1  | not a digit |
| hi    | not a digit |
| 9     | nine        |
| 10    | not a digit |
