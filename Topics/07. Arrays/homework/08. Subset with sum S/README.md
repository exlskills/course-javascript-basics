# Subset with sum S

## Description
We are given an array of integers and a number `S`.
Write a program to find if there exists a subset of the elements of the array that has a sum `S`.

## Sample tests

|       input array                      | S  |     result    |
|:--------------------------------------:|:--:|:-------------:|
| 2, **1**, **2**, 4, 3, **5**, 2, **6** | 14 | yes           |
